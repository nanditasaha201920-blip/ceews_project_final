# Climate–Education Risk Dashboard (Bangladesh)

## Overview
This project identifies climate-vulnerable districts in Bangladesh where education continuity is at risk.

## Objective
To support policymakers in identifying high-risk districts for targeted intervention.

## Methodology
- Combined indicators: flood risk, poverty rate
- Classified districts into Low, Medium, High risk
- Built a simple dashboard using Streamlit

## Tools
- Python
- Pandas
- Streamlit

## Output
Interactive dashboard showing:
- District-level risk
- Risk distribution
- High-risk areas

## Impact
This project demonstrates how data-driven tools can support climate-resilient education planning in Bangladesh.
